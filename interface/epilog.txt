Examples:
    bfcpf.py -c 1XX.X3X.568-XX                  # Brute force CPF, show output on terminal (default) and use OSINT (default)
    bfcpf.py -c XXX.345.567-XX -o no            # Brute force CPF, show output on terminal (default) and don't use OSINT
    bfcpf.py -c XXX.345.567-XX -f yes -o no     # Brute force CPF, save output to file and don't use OSINT
