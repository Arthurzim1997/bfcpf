Examples:
    bfcpf.py -c 1XX.X3X.568-XX                  # Brute force CPF, show output on terminal (default) and use 10 threads (default)
    bfcpf.py -c XXX.345.567-XX -f yes           # Brute force CPF, save output to file and use 10 threads (default)
    bfcpf.py -c XXX.345.567-XX -f yes -t 25     # Brute force CPF, save output to file and use 25 threads
